# 📱 TA Mobile Frontend (React / PWA)

T-Aging 서비스의 모바일 웹앱입니다.
사용자는 모바일 앱을 통해 회원 인증 qr 발급, 주문 이력 조회, 주문 이력 상세 조회(영수증) 등을 확인가능합니다.
또한 중앙 서버(TA Mobile Server)와 REST API로 통신합니다.
